# 🏰 Подземный лабиринт

Консольная игра на C++ с головоломками, где игрок должен выбраться из подземного лабиринта, решая различные задачи.

## 🎮 Об игре

Вы оказываетесь в подземном лабиринте. Ваша цель — найти выход, преодолевая препятствия и решая головоломки. Каждая локация может быть закрыта, и чтобы её открыть, нужно выполнить задание.

### Особенности:
- 🧩 2 типа головоломок (математические задачи и загадки)
- 🗺️ 3 уникальные локации
- 📍 ASCII-графика для визуализации мест
- 🎯 Простая система навигации
-  Полная поддержка русского языка





Удачи в прохождении лабиринта! 🎉
